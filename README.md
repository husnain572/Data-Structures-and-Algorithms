# Data Structures and Algorithms in C++  

This repository contains **complete theory and practical implementations** of Data Structures and Algorithms (DSA) in **C++**. It is a self-learning where I document every topic I study, including explanations and C++ code examples.

- **Sorting Algorithms**  
- **Searching Algorithms**  
- **Arrays**  
- **Linked Lists**  
- **Stacks & Queues**  
- **Trees**  
- **Graphs**  
- **Dynamic Programming**  
- **Other Advanced Topics**  

Each topic includes:  
- **Theory**: Concepts, definitions, and explanations.  
- **Code Implementation**: Practical examples with C++ code.
